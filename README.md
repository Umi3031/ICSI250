# ICSI250
## Зорилго
Энэ хичээл нь програмчлалын суурьтай байхыг шаардахгүй. Загварчлах, тооцооллын арга техник ашиглан шийдэл боловсруулах, шийдлийг програмчлах чадварын үндсийг таниулах, эзэмшүүлэх зорилготой.

## Агуулга
Энэ хичээл нь Python ашиглан програмчлах, загварчлах болон турших, машин сургалтын үндэс гэсэн гурван хэсгээс бүрдэнэ. 
- **Python-аар програмчлах**: Давталт, нөхцөл шалгах, тэмдэгт мөртэй ажиллах; функц, хийсвэрлэл; жагсаалт, хослол, толь гэх мэт өгөгдлийн бүтцийн хэрэглээ, кодын алдаа илрүүлэх, шалгах арга.
- **Загварчлах тооцооллын арга техник**: Граф, сүлжээ, дискрет оновчлол болон тооцооллын туршилт, симуляцын зарим арга зүй (санамсаргүй алхалт, Монте-Карло симуляци, итгэх завсар гэх мэт)-г Python-р програмчлах, туршилтын үр дүнг тайлбарлах.
- **Машин сургалтын үндэс**: Бүлэглэл (clustering), ангилал (classification)-г ойлгох, хэрэглэх дадлага хийх.

## Хичээлээр эзэмших мэдлэг, чадвар, дадал
- Асуудлыг шийдвэрлэхэд тооцоолол ямар үүрэг гүйцэтгэж болох талаар ойлголттой болно.
- Python дээр тооцоолол хийх, тэр дундаа хүснэгтэн өгөгдөлтэй ажиллах боломжтой болно.
- Граф, дискрет оновчлолоор асуудлыг загварчлах, ойлгох чадвартай болно.
- Машин сургалтын зарим үндсэн элементийг ойлгох, хэрэглэх чадвартай болно.

- **Түвшин**: Бакалавр
- **Багц цаг**: 3 бц
- **Ерөнхий эрдмийн хичээл мөн эсэх**: Үгүй
- **Хичээлийн төлөвлөгөөнүүд**:
  - Компьютерын ухаан (Мэргэших хичээл)
  - Мэдээллийн систем (Эрдэм шинжилгээний ажил)
  - Мэдээллийн технологи (Эрдэм шинжилгээний ажил)
  - Эдийн засаг (Мэргэших хичээл)
